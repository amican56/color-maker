# 🧶 編み物カラーデザイナー

編み物作品を作る前に、色の組み合わせを試せるWebアプリです。

## 機能

- ✏️ グリッドをクリックして色を塗る
- 🎨 32色のカラーパレット（日本の伝統色）
- 🖱️ ドラッグで連続塗り
- 🧶 **段ごとモード**: グラニースクエアの段を一括で塗れる
- 🎀 **縁取り機能**: 最外周に幅広の縁取りを表示
- 📐 段の境界線を太線で表示
- 🗑️ 右クリックで色を消す
- 📏 グリッドサイズの変更（5〜50マス）
- 💾 デザインを画像として保存
- 🧹 クリア機能
- 📱 スクロールなしで全機能にアクセス可能

## 使い方

### 通常モード（1マスずつ）
1. 「1マスずつ」を選択
2. カラーパレットから色を選択
3. グリッド上をクリックまたはドラッグして色を塗る

### 段ごとモード（グラニースクエア用）
1. 「段ごと」を選択
2. カラーパレットから色を選択
3. グリッド上の任意の段をクリック → その段全体が塗られる
4. 外側の段から順に色を変えていくと、グラニースクエアのようなデザインが作れる

段の境界線は太線で表示されるので、どこまでが1段かが分かりやすくなっています。

### その他の操作
- 右クリック: 色を消す
- 完成したら「画像として保存」ボタンでダウンロード

## カラーパレット

日本の伝統色を使用しています。各色に色名とカラーコードが表示されます。
- 桜色、紅、藍色、翡翠色など、編み物に映える和の色彩
- 全32色から選択可能
- 参考: [和色大辞典](https://www.colordic.org/w)

## 使い方

1. カラーパレットから色を選択
2. グリッド上をクリックまたはドラッグして色を塗る
3. 右クリックで色を消す
4. 完成したら「画像として保存」ボタンでダウンロード

## デプロイ方法

### GitHub Pagesでの公開

1. GitHubでリポジトリを作成
2. ファイルをアップロード
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/リポジトリ名.git
git push -u origin main
```

3. リポジトリの Settings > Pages で以下を設定：
   - Source: Deploy from a branch
   - Branch: main / (root)
   - Save

4. 数分後、`https://あなたのユーザー名.github.io/リポジトリ名/` でアクセス可能になります

## ローカルでの実行

1. ファイルをダウンロード
2. `index.html` をブラウザで開く

または、VS Codeの「Live Server」拡張機能を使用すると便利です。
